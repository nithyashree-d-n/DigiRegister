# DigiRegister

DigiRegister is an offline-first, web-based digital sales register designed for small institutional stores such as college bookstores.

The project focuses on simplicity, reliability, and ease of use, while supporting offline usage and basic accounting needs.

## Features
- Simple sales entry using +1, +2, +3 quantities
- Cash and UPI payment tracking
- Live daily totals
- Offline-first data storage
- Responsive design for desktop and mobile
- Built using free and open-source technologies

## Tech Stack
- HTML, CSS, JavaScript
- Progressive Web App (PWA)
- Browser-based storage
- Client-side report generation

## Status
This project is under active development and is being built step by step as a learning-focused project.
